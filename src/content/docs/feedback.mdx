This site, the bootstrap instructions, and the agent experience around HDS
improve through your reports. If anything was unclear, broken, or missing
— at any step, on any page —
**[open a GitHub issue on `site-agents`](https://github.com/healthdatasafe/site-agents/issues)**.

Good candidates:

- Your agent misunderstood or ignored part of the instructions.
- A step failed on your machine (include OS, client, and the error).
- You brought a dataset HDS has no item for and had to fall back to notes
  — tell us what the data was, so the data model can grow.
- Wording that confused you (or your agent).

You don't need to be technical: describe what you tried and what happened.
Your agent can draft the issue for you — ask it to.

For bugs in the MCP server itself (crashes, tool errors), the
[`hds-mcp-js` issue tracker](https://github.com/healthdatasafe/hds-mcp-js/issues)
also works — when in doubt, file here on `site-agents` and we'll route it.
